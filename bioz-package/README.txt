BiOz Innovation Website
  =======================

  This package contains two folders:

  1. BUILD/ - The fully compiled static website
     - Open build/index.html directly in any browser — no server needed.
     - All assets (images, CSS, JS) are self-contained in build/assets/.

  2. SOURCE/ - The original React + Vite source code
     - Requires Node.js 18+ and pnpm to run.
     - To install and start locally:
         npm install -g pnpm
         pnpm install
         pnpm run dev
     - Then open http://localhost:5173 in your browser.
